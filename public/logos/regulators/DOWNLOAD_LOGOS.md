# Guide pour télécharger les logos des régulateurs

## Instructions

Pour chaque pays, visitez le site officiel du régulateur et téléchargez le logo au format PNG avec fond transparent.

### Côte d'Ivoire - ANARE
- **Site**: https://anare.ci
- **Fichier à créer**: `anare.png`
- **Instructions**: Visitez anare.ci, faites un clic droit sur le logo dans le header, "Enregistrer l'image sous..."

### Sénégal - CRSE
- **Site**: https://www.crse.sn
- **Fichier à créer**: `crse.png`
- **Instructions**: Visitez crse.sn, le logo est visible dans le header du site

### Ghana - Energy Commission
- **Site**: https://www.energycom.gov.gh
- **Fichier à créer**: `energy-commission-ghana.png`
- **Instructions**: Visitez energycom.gov.gh, le logo est sur la page d'accueil

### Nigeria - NERC

## Format recommandé
- **Format**: PNG avec fond transparent
- **Taille minimale**: 200x200px
- **Résolution**: 72-150 DPI
- **Nom du fichier**: Respecter exactement les noms indiqués ci-dessus

## Alternative rapide
Si vous ne trouvez pas les logos officiels, vous pouvez:
1. Utiliser des outils comme https://www.remove.bg pour enlever le fond
2. Redimensionner avec https://www.iloveimg.com/resize-image
3. Convertir en PNG avec https://cloudconvert.com

Une fois les logos téléchargés, placez-les dans ce dossier (`public/logos/regulators/`) et ils s'afficheront automatiquement dans l'application.
