# Logos des Régulateurs

Ce dossier contient les logos des organismes de régulation du secteur électrique pour chaque pays d'Afrique de l'Ouest.

## Structure des fichiers

Les logos doivent être nommés selon le code pays et placés dans ce dossier :

- `anare.png` - ANARE (Côte d'Ivoire)
- `crse.png` - CRSE (Sénégal)
- `energy-commission-ghana.png` - Energy Commission of Ghana
- `nerc.png` - NERC (Nigeria)
- `area.png` - AREA (Mali)
- `arse-bf.png` - ARSE (Burkina Faso)
- `are-niger.png` - ARE (Niger)
- `arse-togo.png` - ARSE (Togo)
- `are-benin.png` - ARE (Bénin)
- `arse-guinee.png` - ARSE (Guinée)
- `regulator-guinee-bissau.png` - Régulateur (Guinée-Bissau)
- `lerc.png` - LERC (Liberia)
- `ewrc.png` - EWRC (Sierra Leone)
- `pura.png` - PURA (Gambie)
- `regulator-mauritanie.png` - Régulateur (Mauritanie)
- `are-cap-vert.png` - ARE (Cap-Vert)

## Format recommandé

- Format : PNG avec fond transparent
- Taille : 200x200px minimum
- Résolution : 72-150 DPI

## Fallback

Si un logo n'est pas disponible, l'application affichera automatiquement une icône Shield par défaut.
