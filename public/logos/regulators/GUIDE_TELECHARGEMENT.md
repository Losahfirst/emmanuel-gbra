# Guide de Téléchargement des Logos des Régulateurs

Ce guide vous aide à télécharger les logos officiels des organismes de régulation du secteur électrique pour chaque pays d'Afrique de l'Ouest.

## Instructions

1. Téléchargez les logos depuis les sites officiels des régulateurs
2. Renommez-les selon les noms indiqués ci-dessous
3. Placez-les dans ce dossier (`public/logos/regulators/`)
4. Format recommandé : PNG avec fond transparent, taille minimale 200x200px

## Outils Recommandés

Pour optimiser les logos téléchargés :
- **ImageMagick** : Pour convertir et redimensionner
- **GIMP** ou **Photoshop** : Pour retirer le fond et optimiser
- **TinyPNG** : Pour compresser sans perte de qualité

## Format Final

- **Format** : PNG avec fond transparent
- **Taille** : 200x200px minimum (recommandé : 400x400px)
- **Résolution** : 72-150 DPI
- **Poids** : < 100 KB de préférence

## Note

Si un logo n'est pas disponible, l'application affichera automatiquement une icône Shield par défaut. Vous pouvez ajouter les logos progressivement au fur et à mesure de leur disponibilité.
